# 🌦️ WeatherWise AI

WeatherWise AI is a modern weather application with a beautiful glassmorphism UI and RAJ Voice Assistant. It provides real-time weather information, hourly and 5-day forecasts, weather charts, favorite cities, search history, and dark/light mode support.

## Features
- 🌡️ Current Weather
- 📅 5-Day Forecast
- ⏰ Hourly Forecast
- 📊 Weather Charts
- ⭐ Favorite Cities
- 🕒 Search History
- 🌙 Dark/Light Mode
- 🎙️ RAJ Voice Assistant
- 📍 Auto Location Detection
- 📱 Responsive Design

## Technologies Used
- HTML
- CSS
- JavaScript
- OpenWeatherMap API
